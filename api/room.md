## 1. 添加房间
### 1.1 功能描述
往数据库中添加一个房间
### 1.2 请求说明
> 请求方式：POST<br>
> 请求URL ：[room/new](#)

### 1.3 请求参数
| 字段 | 字段类型 | 字段说明        |
| ---- | -------- | --------------- |
| x    | int      | 房间所在的x坐标 |
| y    | int      | 房间所谓的y坐标 |

### 1.4 返回结果
```json  
{
    "errno": 0,
    "errmsg": "",
    "data": {
        "roomId": 1
    }
}
```
### 1.5 返回参数
字段       |字段类型       |字段说明
------------|-----------|-----------
roomId       |int        |创建的房间的数据库编号


### 1.6 错误状态码

参见 [全局响应状态码说明](/#2-全局响应状态码说明)



## 2. 删除房间
### 2.1 功能描述
从数据库中删除一个房间
### 2.2 请求说明
> 请求方式：POST<br>
> 请求URL ：[room/del](#)

### 2.3 请求参数
| 字段   | 字段类型 | 字段说明         |
| ------ | -------- | ---------------- |
| roomId | int      | 房间的数据库编号 |

### 2.4 返回结果

成功操作

```json
{
    "errno": 0,
    "errmsg": "",
    "data": ""
}
```



请求对象不存在

```json
{
    "errno": 1002,
    "errmsg": "invalid roomId"
}
```



请求参数不正确

```json  
{
    "errno": 1001,
    "errmsg": {
        "roomId": "roomId can not be blank"
    }
}
```
### 2.5 返回参数

无

### 2.6 错误状态码
参见 [全局响应状态码说明](/#2-全局响应状态码说明)
