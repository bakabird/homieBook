# 文档使用说明

## 1 返回结果

所有返回结果以 JSON 格式返回；接口返回一共有两种情况：

1. 成功操作返回，范例：

    ```json  
    {
        "errno": 0,
        "errmsg": "",
        "data": ""
    }
    ```

2. 成功返回数据，范例：

    ```json  
    {
        "errno": 0,
        "errmsg": "",
        "data": {    //数据
            "roomId": 1
        }
    }
    ```

3. 错误返回，范例：

    ```json  
    {
      "err_code": "1001",//错误状态码
      "err_msg": "认证失败，请重新登录！"//错误信息
    }
    ```

### 2 全局响应状态码说明

状态码       |说明       
------------|-----------
0        |操作成功    
1001       |参数不合法！    
1002 |操作对象不存在
