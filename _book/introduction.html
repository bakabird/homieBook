1.3.3 返回结果

所有返回结果以 JSON 格式返回；接口返回一共有三种情况：

  1. 操作成功后返回，范例：
    {
"code": "200",//状态码
"msg": "SUCCESS"//信息
}
  2. 成功返回数据，范例：
    {
"data": {//数据
 "name": "minhow",
 "age": "18"
},
"code": "200",//状态码
"msg": "SUCCESS"//信息
}
  3. 错误返回，范例：
    {
"err_code": "1001",//错误状态码
"err_msg": "认证失败，请重新登录！"//错误信息
}

1.3.4 全局响应状态码说明

状态码 说明
200 操作成功
1001 认证失败，请重新登录！
1002 参数不合法！
