1.1 认证管理

1. 登录

1.1 功能描述

提供手机号和密码的登录方式。

1.2 请求说明

请求方式：POST
请求URL ：login

1.3 请求参数

字段 字段类型 字段说明
phone int 手机号
password string 密码

1.4 返回结果

  },
  "code": "200",
  "msg": "SUCCESS"
}

1.5 返回参数

字段 字段类型 字段说明
token string token值

1.6 错误状态码

状态码 说明
3001 其他认证错误信息！
3002 用户不存在！
3003 用户名或密码有误！
